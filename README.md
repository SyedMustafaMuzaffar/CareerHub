# Placement Hub

A comprehensive platform for placement preparation, AI resume building, and readiness tracking.

## Project Structure

- `backend/`: Node.js + Express + MongoDB API
- `frontend/`: Next.js + Tailwind CSS Frontend

## Prerequisites

- Node.js installed
- MongoDB installed and running locally on port 27017

## Getting Started

### Option 1: Run from Root (Recommended)

Open a terminal in the root directory and run:

```bash
npm install
npm run install-all
npm run dev
```

This will install dependencies for both backend and frontend, and then run them simultaneously.

### Option 2: Run Individually

#### 1. Backend Setup

Open a terminal and navigate to the `backend` directory:

```bash
cd backend
npm install
npm start
```

The server should start on `http://localhost:5001`.

### 2. Frontend Setup

Open a new terminal and navigate to the `frontend` directory:

```bash
cd frontend
npm install
npm run dev
```

The application should be running on `http://localhost:3000`.

## Features

- **Placement Prep**: Practice aptitude and coding questions.
- **AI Resume Builder**: Generate professional resumes (Mock AI integration).
- **Readiness Dashboard**: Track your progress and scores.
